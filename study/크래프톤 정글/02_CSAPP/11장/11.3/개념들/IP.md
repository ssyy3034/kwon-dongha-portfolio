## [[IP]]

### 역할

- 인터넷의 네트워크 계층을 구성하는 핵심 프로토콜이다.
    
- 각 호스트를 구분하기 위해 **IP 주소** 체계를 사용하며, [[데이터그램]]이 올바른 목적지에 도달하도록 라우팅한다.
    

### IPv4

- 32비트 주소 체계.
    
- 현재 대부분의 인터넷 트래픽이 IPv4를 기반으로 한다.
    
- 주소는 보통 사람이 읽기 쉽게 “점-십진 표기(dotted-decimal)” 형태로 나타낸다.  
    예: `128.2.194.242`
    

### IPv6

- 128비트 주소 체계.
    
- 주소 공간을 확장하기 위해 도입되었으나, 아직 IPv4가 주로 사용된다.
    

### IP 주소

- 32비트 부호 없는 정수로 표현된다.
    
- 네트워크 전송 시에는 “네트워크 바이트 순서(빅엔디언)” 형식을 따른다.
    
- 프로그램에서는 문자열과 이진 주소 간 변환을 위해 `inet_pton()` / `inet_ntop()` 같은 함수를 사용한다.
